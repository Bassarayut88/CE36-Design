แผ่นที่: 1 / 1

ขนาดฐาน (m):
ความหนา H: m
ดินรับน้ำหนักปลอดภัย: t/m²
ระยะหุ้ม: cm
Dead Load (PD): ton
Live Load (PL): ton
เหล็กเสริม:
1. ตรวจสอบแรงดันดิน (Soil Bearing)
น้ำหนักฐานราก (Wf) = Lx × Ly × H × 2400
น้ำหนักใช้งานรวม (ΣP) = Wf + PD + PL
แรงดันดินใช้งาน (qp) = ΣP / A
ตรวจสอบ (qp ≤ fb)
2. น้ำหนักบรรทุกประลัย (Ultimate Loads)
น้ำหนักประลัย (Pu) = 1.4(PD+Wf) + 1.7(PL)
แรงดันดินประลัย (qu) = Pu / A
3. ตรวจสอบแรงเฉือน (Shear Design)
ความลึกประสิทธิผลเฉลี่ย (d_avg)
แรงเฉือนคาน ทิศ X (Vux) ≤ φVcx
แรงเฉือนคาน ทิศ Y (Vuy) ≤ φVcy
แรงเฉือนทะลุ (Vup) ≤ φVcp
4. ออกแบบเหล็กเสริม (Flexural Design)
โมเมนต์ประลัย (Mux / Muy)
เหล็กเสริมขั้นต่ำ (As_min) = 0.0018 Ag
เหล็กเสริมที่ใช้ (As_provided)
กำลังรับโมเมนต์ (φMn ≥ Mu)
